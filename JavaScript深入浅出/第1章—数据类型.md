### 第一章 数据类型
#### 1. 六种数据类型
+ eg：
```javascript
    var num = 32;
    num = "this is a string";   //这样是合法的
    32 + 32                     //64
    "32" + 32                   //3232
    "32" - 32                   //0
```

JavaScript中有哪些数据类型呢？

我们把比价基础简单的类型成为*原始类型*，也叫*基元类型*，示例如下：

+ number
+ string
+ boolean
+ null
+ undefined

null和undefined是两种比较特殊的原始类型

除了这五种原始类型外，剩下的一种叫做对象(object),我们此时会比较好奇，数组(Array)和函数(Function)哪里去了？

在JavaScript中，函数(Function)，数组(Array)，日期(Date)等都是对象。
